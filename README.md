# Aniset KWGT - Website 

This repository contains the source code for the official landing page of **Aniset KWGT**, a premium anime-themed widget pack for Android devices. The website is built with React, Vite, and Tailwind CSS to showcase the app's features, gallery, and user reviews.

![Aniset App Screenshot](https://raw.githubusercontent.com/satyakiran29/aniset_web/main/public/screenshots/1%20(1).webp)

## About Aniset KWGT

Aniset KWGT is an Android application that provides a collection of over 100 high-quality, customizable widgets for the [KWGT Kustom Widget Maker](https://play.google.com/store/apps/details?id=org.kustom.widget). It allows users to transform their home screens with designs inspired by popular anime series.

### Key Features of the App
- **100+ Anime Widgets:** A vast and growing collection of premium widgets.
- **Fully Customizable:** Easily adjust colors, sizes, and elements to match your wallpaper and theme.
- **Regular Updates:** New widgets are added frequently to keep your setups fresh.
- **Lightweight & Optimized:** Designed for minimal battery usage and optimal performance.
- **Responsive:** Widgets look great on any screen size or aspect ratio.

## About This Website

This landing page serves as the central hub for Aniset KWGT. It provides users with all the necessary information, including an installation guide, feature overview, user testimonials, and direct contact with the developer.

### Tech Stack
-   **Framework:** [React](https://reactjs.org/)
-   **Build Tool:** [Vite](https://vitejs.dev/)
-   **Styling:** [Tailwind CSS](https://tailwindcss.com/)
-   **Animations:** [Framer Motion](https://www.framer.com/motion/)
-   **Routing:** [React Router](https://reactrouter.com/)
-   **Icons:** [Lucide React](https://lucide.dev/)

### Website Features
-   **Modern UI/UX:** Clean, visually appealing design with a focus on user experience.
-   **Dark & Light Mode:** A theme-switcher that respects user system preference and saves the choice in local storage.
-   **Responsive Design:** Fully optimized for seamless viewing on both desktop and mobile devices.
-   **Component-Based:** Built with a modular and maintainable component structure.
-   **Dynamic Content:** Features sections for community links, FAQs, developer information, and more.

## Getting Started

To run this project locally, follow these steps:

**1. Clone the repository:**
```bash
git clone https://github.com/satyakiran29/aniset_web.git
```

**2. Navigate to the project directory:**
```bash
cd aniset_web
```

**3. Install dependencies:**
```bash
npm install
```

**4. Run the development server:**
```bash
npm run dev
```
The application will be available at `http://localhost:5173`.
